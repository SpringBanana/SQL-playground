## Задания:

### Задание 1

Вывести **уникальные названия городов** из таблицы `city`.  

**Требования к результату:**
- В результирующей таблице должен быть один столбец: название города.

### Задание 2

Доработать запрос из задания 1 так, чтобы он выводил только те города,  
названия которых:

- начинаются на букву `L`;
- заканчиваются на букву `a`;
- **не содержат пробелов** внутри строки.

**Требования к результату:**
- В результирующей таблице должен быть один столбец: название города.

### Задание 3

Получить из таблицы платежей за прокат фильмов информацию о платежах, которые:

- выполнены в промежуток **с 17 июня 2005 года по 19 июня 2005 года включительно**;
- имеют **сумму платежа больше 1.00**.  

Результат отсортировать **по дате платежа** (от более ранних к более поздним).

**Требования к результату:**
- идентификатор платежа;
- размер платежа;
- дата платежа.

### Задание 4

Вывести информацию о **10 последних платежах** за прокат фильмов.

**Требования к результату:**
- в результирующей таблице должны быть **все столбцы** из таблицы платежей.

### Задание 5

Вывести следующую информацию по покупателям:

- фамилия и имя в одной колонке через пробел;
- электронная почта;
- длина значения поля `email`;
- дата **последнего обновления** записи о покупателе (без времени).

**Требования к результату:**

- В результирующей таблице должны быть только столбцы, указанные в условии.
- Каждой колонке задать **алиас на русском языке**.

### Задание 6

Вывести одним запросом только **активных покупателей**, имена которых `KELLY` или `WILLIE`.  

Все буквы в фамилии и имени должны быть переведены **из верхнего регистра в нижний**.

**Требования к результату:**

- имя покупателя в нижнем регистре;
- фамилия покупателя в нижнем регистре;
- флаг активности.
