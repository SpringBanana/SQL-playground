#  SQL Homeworks

В этой папке собраны мои **домашние задания по SQL**, выполненные на учебной базе данных DVD Rental и других схемах.  
Каждая домашка оформлена как небольшой мини‑проект с отдельным `README.md` и файлом с решениями в `*.sql`.

## Навигация по домашкам

- [`DVD-rental_basics`](./DVD-rental_basics/README.md) — базовые запросы, агрегации, JOIN, оконные функции и работа с DVD Rental.
- [`Final`](./Final/README.md) — итоговая работа по SQL с комплексными аналитическими запросами, рекурсивными CTE и материализованными представлениями на базе данных строительной компании.

## Как читать домашки

1. Перейти в интересующую папку
2. Открыть локальный `README.md`, чтобы посмотреть:
   - цели домашнего задания;
   - формулировки задач;
   - ссылку на файл с решениями (`HW*_Solutions.sql` или `Final_Solutions.sql`).
3. По желанию открыть SQL‑файлы в клиенте БД и прогнать запросы на своей копии базы.

Домашки оформлены так, чтобы их было удобно использовать и как **конспект по SQL**, и как часть портфолио.
