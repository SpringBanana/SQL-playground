# HW2 — Агрегация, группировка и JOIN

## Цели домашнего задания

- закрепить на практике знания по функциям агрегации и группировке строк;
- научиться фильтровать сгруппированные строки;
- закрепить навыки использования методов соединения таблиц с помощью разных вариаций `JOIN`.

## Задания:

### Задание 1

Вывести для каждого покупателя его адрес, город и страну проживания.

**Требования к результату:**

- Имя пользователя;
- Фамилия пользователя;
- Адрес;
- Город;
- Страна.

---

### Задание 2.1

С помощью SQL-запроса посчитать для каждого магазина количество его покупателей.

**Требования к результату:**

- Идентификатор магазина;
- Количество прикреплённых пользователей.

---

### Задание 2.2

Доработать запрос из задания 2.1 и вывести только те магазины, у которых количество покупателей **больше 300**.  
Для решения использовать фильтрацию по сгруппированным строкам с функцией агрегации.

**Требования к результату:**

- Идентификатор магазина;
- Количество прикреплённых пользователей.

---

### Задание 2.3

Доработать запрос из задания 2.2, добавив в него информацию:

- о городе нахождения магазина;
- о продавце (сотруднике), который работает в этом магазине.

**Требования к результату:**

- Фамилия и имя сотрудника в виде одного значения;
- Идентификатор магазина;
- Город нахождения магазина;
- Количество прикреплённых пользователей.

---

### Задание 3

Вывести **топ-5 покупателей**, которые взяли в аренду за всё время наибольшее количество фильмов.

**Требования к результату:**

- Фамилия и имя пользователя в виде одного значения;
- Количество арендованных фильмов.

---

### Задание 4

Посчитать для каждого покупателя следующие 4 аналитических показателя:

- количество взятых в аренду фильмов;
- общую стоимость платежей за аренду всех фильмов (значение округлить до целого числа);
- минимальное значение платежа за аренду фильма;
- максимальное значение платежа за аренду фильма.

**Требования к результату:**

- Фамилия и имя пользователя в виде одного значения;
- Количество арендованных фильмов;
- Округлённая сумма платежей;
- Минимальный платеж;
- Максимальный платеж.

---

### Задание 5

Используя данные из таблицы городов, составить **все возможные пары городов** так, чтобы в результате **не было пар с одинаковыми названиями городов**.  
Задание необходимо выполнить, используя **декартово произведение**.

**Требования к результату:**

- Два столбца с названиями городов.

---

### Задание 6

Используя данные из таблицы `rental` о дате выдачи фильма в аренду (`rental_date`) и дате возврата (`return_date`), вычислить для каждого покупателя **среднее количество дней**, за которые он возвращает фильмы, округлённое до сотых.

**Требования к результату:**

- Фамилия и имя пользователя в виде одного значения;
- Среднее количество дней с учётом округления.
