# SQL Homeworks & Side Projects

Этот репозиторий — моя коллекция домашних заданий и сторонних проектов по **SQL**.  
Здесь я собираю примеры запросов, мини‑проекты и эксперименты с учебными и реальными данными: от базовых SELECT до сложной аналитики с CTE и оконными функциями.

## Структура репозитория

Репозиторий делится на два основных блока:

- [`homeworks/`](./homeworks/) — домашние задания  
- [`projects/`](./projects/) — самостоятельные аналитические проекты на SQL
