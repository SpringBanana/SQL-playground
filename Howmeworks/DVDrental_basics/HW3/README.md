## Задания:

### Общие условия

**База данных:**

- если подключение к локальному серверу — создать новую схему и в ней создать таблицы.

### Задание 1

Спроектировать базу данных, содержащую три справочника:

- язык (например: английский, французский и т. п.);
- народность (например: славяне, англосаксы и т. п.);
- страны (например: Россия, Германия и т. п.).

А также две таблицы со связями:

- связь язык–народность;
- связь народность–страна;

причём отношения — **многие ко многим** (аналогично таблице `film_actor` в DVD Rental).

**Требования к таблицам-справочникам:**

- наличие ограничений первичных ключей;
- идентификатор сущности должен присваиваться автоинкрементом;
- наименования сущностей не должны содержать `NULL`-значений;
- не должны допускаться дубликаты в названиях сущностей.

**Требования к таблицам со связями:**

- наличие ограничений первичных и внешних ключей.

**Ответ на задание должен включать:**

- SQL-запросы создания всех таблиц;
- SQL-запросы по добавлению **минимум 5 строк данных** в каждую таблицу.

## Дополнительная часть

### Задание 1

Создать новую таблицу `film_new` со следующими полями:

- `film_name` — название фильма, тип данных `varchar(255)`, ограничение `NOT NULL`;
- `film_year` — год выпуска фильма, тип данных `integer`, условие, что значение **должно быть больше 0**;
- `film_rental_rate` — стоимость аренды фильма, тип данных `numeric(4,2)`, значение по умолчанию `0.99`;
- `film_duration` — длительность фильма в минутах, тип данных `integer`, ограничение `NOT NULL` и условие, что значение **должно быть больше 0**.

Если работа ведётся в облачной базе, перед названием таблицы указать имя своей схемы.

---

### Задание 2

Заполнить таблицу `film_new` данными с помощью одного SQL-запроса, в котором колонкам соответствуют массивы значений:

- `film_name` — `array['The Shawshank Redemption', 'The Green Mile', 'Back to the Future', 'Forrest Gump', 'Schindler’s List']`;
- `film_year` — `array[1994, 1999, 1985, 1994, 1993]`;
- `film_rental_rate` — `array[2.99, 0.99, 1.99, 2.99, 3.99]`;
- `film_duration` — `array[142, 189, 116, 142, 195]`.

### Задание 3

Обновить стоимость аренды всех фильмов в таблице `film_new` с учётом информации, что стоимость аренды **поднялась на 1.41** для каждого фильма.
